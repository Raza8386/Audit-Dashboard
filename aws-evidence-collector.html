AWS SOC 2 Evidence Collector

Automated Python script that collects compliance evidence from AWS services and exports audit-ready reports mapped to SOC 2 Trust Service Criteria.

Python 3.11+ Read-only · No destructive ops JSON & CSV export boto3 · AWS SDK CC6.1 · CC6.2 · CC6.3 · CC6.7 · CC7.2 · CC7.3

✅

Use this script when…

  • Preparing for an initial SOC 2 audit
  • Collecting quarterly evidence for ongoing SOC 2 compliance
  • Performing gap analysis against SOC 2 requirements
  • Responding to auditor evidence requests
  • Learning AWS SOC 2 evidence requirements
⛔

Do NOT use for…

  • Continuous monitoring (use AWS Audit Manager)
  • Real-time security alerting (use AWS Security Hub)
  • Comprehensive security assessment
  • Multi-account automated collection at scale

Install Python dependencies

# Recommended: create a virtual environment first

pip install -r requirements.txt

requirements.txt contains: boto3>=1.35.0

Configure AWS credentials

# Configure a named profile
aws configure --profile soc2-evidence

# Verify it works
aws sts get-caller-identity --profile soc2-evidence
4

Run a test collection

python collector.py --profile soc2-evidence --output ./test-run

IAM Permissions

✅
The script is 100% read-only. It never creates, modifies, or deletes any AWS resources. All API calls are logged in CloudTrail.

The script needs these read-only permissions across 6 AWS services:

ServiceActions RequiredPurpose
IAMListUsers ListMFADevices ListAccessKeys ListGroupsForUser ListAttachedUserPoliciesCC6.1 – CC6.3 evidence
S3ListAllMyBuckets GetBucketEncryptionCC6.7 — bucket encryption
RDSDescribeDBInstancesCC6.7 — DB encryption
EC2DescribeVolumesCC6.7 — EBS encryption
CloudTrailDescribeTrails GetTrailStatus GetEventSelectorsCC7.2 — logging evidence
CloudWatch / LogsDescribeAlarms DescribeLogGroupsCC7.3 — monitoring evidence
STSGetCallerIdentityIdentify account for evidence metadata

Setup a dedicated IAM user for evidence collection

# 1. Create the IAM policy from the template
aws iam create-policy \
  --policy-name SOC2EvidenceCollectorReadOnly \
  --policy-document file://iam-policy.json

# 2. Create a dedicated IAM user
aws iam create-user --user-name soc2-evidence-collector

# 3. Attach the policy (replace ACCOUNT_ID)
aws iam attach-user-policy \
  --user-name soc2-evidence-collector \
  --policy-arn arn:aws:iam::ACCOUNT_ID:policy/SOC2EvidenceCollectorReadOnly

# 4. Create access keys
aws iam create-access-key --user-name soc2-evidence-collector

# 5. Configure a local AWS CLI profile
aws configure --profile soc2-evidence

ℹ️
Multi-region note: IAM evidence is global, but S3 encryption, CloudTrail, and CloudWatch evidence is region-specific. Run the script once per region if you have resources spread across multiple regions.

collector.py — Full Source

collector.py
Python 3.11+ 
#!/usr/bin/env python3
"""
AWS SOC 2 Evidence Collector
Collects compliance evidence from AWS for SOC 2 audit preparation.

Usage:
    python collector.py --profile <aws-profile> --output <output-dir> --format <json|csv|both>
"""

import boto3
import json
import csv
import argparse
import os
from datetime import datetime
from typing import Dict, List, Any


class AWSEvidenceCollector:
    """Collects SOC 2 compliance evidence from AWS services."""

    def __init__(self, profile: str = None, region: str = None):
        session_args = {}
        if profile:
            session_args['profile_name'] = profile
        session_args['region_name'] = region or 'us-east-1'
        try:
            self.session = boto3.Session(**session_args)
            self.region = self.session.region_name
            sts = self.session.client('sts')
            self.account_id = sts.get_caller_identity()['Account']
        except Exception as e:
            raise RuntimeError(f"Failed to initialize AWS session: {e}")
        self.timestamp = datetime.utcnow().isoformat() + 'Z'

    def collect_iam_evidence(self) -> Dict[str, Any]:
        """CC6.1, CC6.2, CC6.3 — IAM user access controls."""
        print("Collecting IAM evidence...")
        iam = self.session.client('iam')
        evidence = {
            'collection_timestamp': self.timestamp,
            'account_id': self.account_id,
            'control_mapping': ['CC6.1', 'CC6.2', 'CC6.3'],
            'evidence_type': 'IAM User Access Report',
            'users': []
        }
        paginator = iam.get_paginator('list_users')
        for page in paginator.paginate():
            for user in page['Users']:
                name = user['UserName']
                try:
                    mfa      = iam.list_mfa_devices(UserName=name)['MFADevices']
                    keys     = iam.list_access_keys(UserName=name)['AccessKeyMetadata']
                    groups   = iam.list_groups_for_user(UserName=name)['Groups']
                    policies = iam.list_attached_user_policies(UserName=name)['AttachedPolicies']
                    pwd_used = user.get('PasswordLastUsed')
                    evidence['users'].append({
                        'username':         name,
                        'user_id':          user['UserId'],
                        'arn':              user['Arn'],
                        'created_date':     user['CreateDate'].isoformat(),
                        'password_last_used': pwd_used.isoformat() if pwd_used else None,
                        'mfa_enabled':      len(mfa) > 0,
                        'mfa_devices':      [{'serial': d['SerialNumber']} for d in mfa],
                        'access_keys':      [{'key_id': k['AccessKeyId'], 'status': k['Status'], 'created': k['CreateDate'].isoformat()} for k in keys],
                        'groups':           [g['GroupName'] for g in groups],
                        'attached_policies': [p['PolicyName'] for p in policies],
                    })
                except Exception as e:
                    print(f"  Warning: Could not collect data for user {name}: {e}")
        evidence['summary'] = {
            'total_users':        len(evidence['users']),
            'users_with_mfa':     sum(1 for u in evidence['users'] if u['mfa_enabled']),
            'users_without_mfa':  sum(1 for u in evidence['users'] if not u['mfa_enabled']),
            'users_with_keys':    sum(1 for u in evidence['users'] if u['access_keys']),
        }
        print(f"  Collected evidence for {evidence['summary']['total_users']} IAM users")
        return evidence

    def collect_encryption_evidence(self) -> Dict[str, Any]:
        """CC6.7 — Encryption at rest for S3, RDS, EBS."""
        print("Collecting encryption evidence...")
        evidence = {
            'collection_timestamp': self.timestamp,
            'account_id': self.account_id,
            'region': self.region,
            'control_mapping': ['CC6.7'],
            's3_buckets': [], 'rds_instances': [], 'ebs_volumes': []
        }
        # S3
        s3 = self.session.client('s3')
        for b in s3.list_buckets()['Buckets']:
            try:
                enc  = s3.get_bucket_encryption(Bucket=b['Name'])
                algo = enc['ServerSideEncryptionConfiguration']['Rules'][0]['ApplyServerSideEncryptionByDefault']['SSEAlgorithm']
                evidence['s3_buckets'].append({'bucket_name': b['Name'], 'encrypted': True, 'encryption_type': algo})
            except:
                evidence['s3_buckets'].append({'bucket_name': b['Name'], 'encrypted': False, 'encryption_type': None})
        # RDS
        rds = self.session.client('rds')
        for page in rds.get_paginator('describe_db_instances').paginate():
            for inst in page['DBInstances']:
                evidence['rds_instances'].append({
                    'db_instance_identifier': inst['DBInstanceIdentifier'],
                    'engine':   inst['Engine'],
                    'encrypted': inst.get('StorageEncrypted', False),
                    'kms_key_id': inst.get('KmsKeyId'),
                })
        # EBS
        ec2 = self.session.client('ec2')
        for page in ec2.get_paginator('describe_volumes').paginate():
            for vol in page['Volumes']:
                evidence['ebs_volumes'].append({
                    'volume_id': vol['VolumeId'],
                    'size_gb':   vol['Size'],
                    'encrypted': vol['Encrypted'],
                    'kms_key_id': vol.get('KmsKeyId'),
                })
        evidence['summary'] = {
            's3_encrypted':   sum(1 for b in evidence['s3_buckets']    if b['encrypted']),
            'rds_encrypted':  sum(1 for r in evidence['rds_instances'] if r['encrypted']),
            'ebs_encrypted':  sum(1 for v in evidence['ebs_volumes']    if v['encrypted']),
        }
        return evidence

    def collect_cloudtrail_evidence(self) -> Dict[str, Any]:
        """CC7.2 — CloudTrail logging configuration."""
        print("Collecting CloudTrail evidence...")
        ct = self.session.client('cloudtrail')
        trails = []
        for t in ct.describe_trails()['trailList']:
            status = ct.get_trail_status(Name=t['Name'])
            trails.append({
                'name':                      t['Name'],
                's3_bucket':                 t.get('S3BucketName'),
                'is_multi_region':           t.get('IsMultiRegionTrail', False),
                'is_logging':                status.get('IsLogging', False),
                'log_file_validation_enabled': t.get('LogFileValidationEnabled', False),
            })
        return {'trails': trails, 'control_mapping': ['CC7.2'],
                'summary': {'total_trails': len(trails),
                            'logging_enabled': sum(1 for t in trails if t['is_logging'])}}

    def collect_cloudwatch_evidence(self) -> Dict[str, Any]:
        """CC7.3 — CloudWatch alarms and log groups."""
        print("Collecting CloudWatch evidence...")
        cw   = self.session.client('cloudwatch')
        logs = self.session.client('logs')
        alarms, log_groups = [], []
        for page in cw.get_paginator('describe_alarms').paginate():
            for a in page['MetricAlarms']:
                alarms.append({'alarm_name': a['AlarmName'], 'state': a['StateValue'],
                               'actions_enabled': a.get('ActionsEnabled', False)})
        for page in logs.get_paginator('describe_log_groups').paginate():
            for lg in page['logGroups']:
                log_groups.append({'name': lg['logGroupName'], 'retention_days': lg.get('retentionInDays')})
        return {'alarms': alarms, 'log_groups': log_groups, 'control_mapping': ['CC7.3'],
                'summary': {'total_alarms': len(alarms), 'total_log_groups': len(log_groups)}}

    def export_json(self, evidence: Dict, output_dir: str):
        os.makedirs(output_dir, exist_ok=True)
        safe_ts = self.timestamp.replace(':', '-').replace('.', '-')
        path = os.path.join(output_dir, f'aws_evidence_{safe_ts}.json')
        with open(path, 'w') as f:
            json.dump(evidence, f, indent=2, default=str)
        print(f"JSON exported to: {path}")
        return path


def main():
    parser = argparse.ArgumentParser(description='AWS SOC 2 Evidence Collector')
    parser.add_argument('--profile', help='AWS profile name')
    parser.add_argument('--region',  default='us-east-1')
    parser.add_argument('--output',  default='./evidence')
    parser.add_argument('--format',  choices=['json', 'csv', 'both'], default='json')
    args = parser.parse_args()

    collector = AWSEvidenceCollector(profile=args.profile, region=args.region)
    print(f"Authenticated as Account: {collector.account_id}")

    evidence = {
        'metadata':   {'collection_timestamp': collector.timestamp, 'account_id': collector.account_id,
                       'region': collector.region, 'collector_version': '1.0'},
        'iam':        collector.collect_iam_evidence(),
        'encryption': collector.collect_encryption_evidence(),
        'logging':    collector.collect_cloudtrail_evidence(),
        'monitoring': collector.collect_cloudwatch_evidence(),
    }

    if args.format in ['json', 'both']:
        collector.export_json(evidence, args.output)
    print("Evidence collection complete!")


if __name__ == '__main__':
    exit(main())

Troubleshooting

❌ "Failed to initialize AWS session" or "Unable to locate credentials"▶
Cause: AWS credentials not configured or profile not found.

Fix: Run aws configure --profile your-profile or check ~/.aws/credentials contains the specified profile. Try without --profile to use default credentials.
❌ "AccessDenied when calling X operation"▶
Cause: IAM user/role lacks a required permission.

Fix: Verify the IAM policy from iam-policy.json is attached to your user/role. Check with aws iam list-attached-user-policies --user-name your-user. The error message names the exact action needed.
⚠️ "Warning: Could not collect X evidence"▶
Cause: Service not enabled, region has no resources, or partial permissions.

Impact: Script continues — this is informational only. Check if the service is enabled in your account and that the region has resources (e.g. aws s3 ls).
❌ "No module named 'boto3'"▶
Fix: Run pip install -r requirements.txt. If using a virtual environment, activate it first: source venv/bin/activate (Linux/Mac) or venv\Scripts\activate (Windows).
🐢 Script runs slowly on large accounts▶
Cause: Account has thousands of IAM users, S3 buckets, or EBS volumes. The script uses pagination — this is expected.

Tips: Run during off-hours. For accounts with 10,000+ resources, expect 5–10 minutes. Consider AWS Audit Manager for continuous collection at scale.
⚠️ "Rate exceeded" errors▶
Cause: Too many API calls in a short period (AWS throttling).

Fix: Wait a few minutes and retry. Run the script less frequently. Use AWS Audit Manager for continuous compliance monitoring instead.

Limitations

⚠️
This script does NOT collect: application logs, VPC Flow Logs, AWS Config rules, Security Hub findings, GuardDuty findings, or IAM Access Analyzer findings. Use dedicated AWS APIs or AWS Audit Manager for those.
